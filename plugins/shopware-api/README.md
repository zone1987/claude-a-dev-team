# shopware-api

Shopware 6 APIs: Admin API (OAuth client_credentials/password), Store API (sw-access-key/sw-context-token),
Sync API — Endpunkte, Authentifizierung, Requests/Responses, Parameter/Header, Fehlerformat.
Plus Introspektion der projekt-eigenen OpenAPI-Spec (alle Endpunkte, immer versionsaktuell).

Teil der Shopware-Entwicklungsbibliothek im Marketplace **claude-a-dev-team**.
