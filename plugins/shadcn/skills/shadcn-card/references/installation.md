# Card — Installation

## CLI (recommended)

```bash
npx shadcn@latest add card
```

## Manual

1. Copy the source from `references/source.md` into `components/ui/card.tsx`.
2. Update the import paths to match your project setup.

No additional npm dependencies are required — the Card component is pure HTML/Tailwind with no external primitives.
