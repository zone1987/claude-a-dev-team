# shadcn-drawer

The `Drawer` component is a slide-in panel built on [Vaul](https://github.com/emilkowalski/vaul). Supports `top`, `right`, `bottom`, and `left` directions via the `direction` prop. Commonly used as a responsive alternative to `Dialog` on mobile.

## Quick start

```bash
npx shadcn@latest add drawer
```
