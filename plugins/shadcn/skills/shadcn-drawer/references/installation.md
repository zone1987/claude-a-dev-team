# Drawer — Installation

## CLI (recommended)

```bash
npx shadcn@latest add drawer
```

## Manual

1. Install dependency:

```bash
npm install vaul
```

2. Copy `components/ui/drawer.tsx` into your project (see `source.md`).
3. Update import paths to match your project.

## About

Drawer is built on top of [Vaul](https://github.com/emilkowalski/vaul) by [emilkowalski](https://twitter.com/emilkowalski). No Radix UI / Base UI dependency — Vaul handles everything.
