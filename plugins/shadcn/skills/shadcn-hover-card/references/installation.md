# HoverCard — Installation

## CLI (recommended)

```bash
npx shadcn@latest add hover-card
```

## Manual — Radix UI (new-york-v4 default)

1. Install dependency:

```bash
npm install radix-ui
```

2. Copy `components/ui/hover-card.tsx` into your project (see `source.md`).
3. Update import paths to match your project.

## Manual — Base UI

1. Install dependency:

```bash
npm install @base-ui/react
```

2. Copy the Base UI variant into your project.
