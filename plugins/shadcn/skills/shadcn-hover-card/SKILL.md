# shadcn-hover-card

The `HoverCard` displays rich preview content when the user hovers over a trigger element. Built on Radix UI. Supports positioning via `side`/`align`, open/close delays, and RTL.

## Quick start

```bash
npx shadcn@latest add hover-card
```
