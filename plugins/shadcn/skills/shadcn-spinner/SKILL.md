# shadcn/ui Spinner

An indicator that can be used to show a loading state.
Uses `Loader2Icon` from lucide-react with `animate-spin` by default.
Can be customized with any SVG icon.

## Quick Reference

- **Install**: `npx shadcn@latest add spinner`
- **No external dep** (uses lucide-react)
- **Exports**: `Spinner`
- **Customizable**: swap icon, adjust size with `size-*` classes, color with `text-*`
