# shadcn-input-otp

The `InputOTP` component provides an accessible one-time password input built on [input-otp](https://github.com/guilhermerodz/input-otp). Slots are rendered individually and grouped. Supports copy-paste, numeric/alphanumeric patterns, controlled state, and RTL.

## Quick start

```bash
npx shadcn@latest add input-otp
```
