# InputOTP — Installation

## CLI (recommended)

```bash
npx shadcn@latest add input-otp
```

## Manual

1. Install dependency:

```bash
npm install input-otp
```

2. Copy `components/ui/input-otp.tsx` into your project (see `source.md`).
3. Update import paths to match your project.

## About

InputOTP is built on top of [input-otp](https://github.com/guilhermerodz/input-otp) by [@guilherme_rodz](https://twitter.com/guilherme_rodz). Full docs at [input-otp.rodz.dev](https://input-otp.rodz.dev).
