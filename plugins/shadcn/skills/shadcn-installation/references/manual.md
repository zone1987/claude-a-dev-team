# shadcn/ui — Manual Installation

Use this guide when your framework is not supported by `npx shadcn@latest init -t`.

## Step 1 — Add Tailwind CSS

Follow the official Tailwind CSS installation: https://tailwindcss.com/docs/installation

## Step 2 — Add dependencies

```bash
npm install shadcn class-variance-authority clsx tailwind-merge lucide-react tw-animate-css
```

## Step 3 — Configure import aliases

### Option A: tsconfig.json paths
```json
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["./*"]
    }
  }
}
```

### Option B: package.json#imports
```json
{
  "imports": {
    "#components/*": "./src/components/*.tsx",
    "#lib/*": "./src/lib/*.ts",
    "#hooks/*": "./src/hooks/*.ts"
  }
}
```
```json
{
  "compilerOptions": {
    "moduleResolution": "bundler",
    "resolvePackageJsonImports": true
  }
}
```
