# shadcn/ui Select

Displays a list of options for the user to pick from — triggered by a button.
Available in Radix UI and Base UI variants.

## Quick Reference

- **Install**: `npx shadcn@latest add select`
- **Radix dep**: `radix-ui` | **Base dep**: `@base-ui/react`
- **Exports**: `Select`, `SelectContent`, `SelectGroup`, `SelectItem`, `SelectLabel`,
  `SelectScrollDownButton`, `SelectScrollUpButton`, `SelectSeparator`, `SelectTrigger`, `SelectValue`
