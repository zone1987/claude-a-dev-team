# Popover — Installation

## CLI

```bash
npx shadcn@latest add popover
```

## Manual

1. Install dependency:

```bash
npm install radix-ui
# or specifically:
npm install @radix-ui/react-popover
```

2. Copy the source into `components/ui/popover.tsx`.
3. Update import paths.
