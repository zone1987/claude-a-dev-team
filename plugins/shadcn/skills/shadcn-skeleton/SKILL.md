# shadcn/ui Skeleton

Use to show a placeholder while content is loading. Pure Tailwind, no external dependency.

## Quick Reference

- **Install**: `npx shadcn@latest add skeleton`
- **No dependency** (pure `animate-pulse` Tailwind)
- **Exports**: `Skeleton`
