# shadcn/ui — Figma

Community-maintained Figma files recreating shadcn/ui components with
customisable props, typography, and icons.

Note: Figma files are contributed by the community, not the core shadcn/ui team.

## Free resources

- **shadcn/ui components** by Sitsiilia Bergmann
  https://www.figma.com/community/file/1342715840824755935
  Well-structured, regularly maintained component library.

- **shadcn/ui design system** by Pietro Schirano
  https://www.figma.com/community/file/1203061493325953101
  Design companion, each component painstakingly matched to code.

## Paid resources

- **shadcn/ui kit** by Matt Wierzbicki — https://shadcndesign.com
  Premium, always up-to-date UI kit, optimised for design-to-dev handoff.

- **shadcncraft Design System** — https://shadcncraft.com
  Production-ready kit with Pro React blocks, 1:1 Figma alignment,
  tweakcn theming, AI workflows, Figma to React export.

- **shadcn/studio UI Kit** — https://shadcnstudio.com/figma
  550+ blocks, 10+ templates, 20+ themes, AI design-to-code tool.

- **Shadcnblocks.com** — https://www.shadcnblocks.com
  500+ pro blocks, shadcn theme variables, light/dark mode, Figma MCP.

- **Obra shadcn/ui Pro** — https://shadcn.obra.studio/products/obra-shadcn-ui-pro
  Best designer experience in Figma; variable consistency with shadcn,
  custom components, Pro blocks, design-to-code plugin.
