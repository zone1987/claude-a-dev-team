# shadcn/ui Slider

An input where the user selects a value from within a given range.
Supports single value, range (two thumbs), multiple thumbs, and vertical orientation.

## Quick Reference

- **Install**: `npx shadcn@latest add slider`
- **Radix dep**: `radix-ui` | **Base dep**: `@base-ui/react`
- **Exports**: `Slider`
- Single value: `defaultValue={[50]}`, Range: `defaultValue={[20, 80]}`
