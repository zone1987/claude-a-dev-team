# shadcn/ui Switch

A control that allows the user to toggle between checked and not checked. Built on Radix UI Toggle primitive (new-york-v4 style) or Base UI Switch.

## Quick Reference

- **Install**: `npx shadcn@latest add switch`
- **Radix deps**: `radix-ui`
- **Base deps**: `@base-ui/react`
- **Exports**: `Switch`
- **Sizes**: `default` (1.15rem x 2rem), `sm` (0.875rem x 1.5rem)
