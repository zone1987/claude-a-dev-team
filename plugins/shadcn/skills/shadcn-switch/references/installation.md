# Switch — Installation

## CLI (recommended)

```bash
npx shadcn@latest add switch
```

## Manual — Radix

Install dependency:

```bash
npm install radix-ui
```

Copy `components/ui/switch.tsx` (see source.md).
Update import paths to match your project.

## Manual — Base UI

Install dependency:

```bash
npm install @base-ui/react
```

Copy the Base UI variant of `components/ui/switch.tsx` (see source.md).
Update import paths to match your project.

## Usage

```tsx
import { Switch } from "@/components/ui/switch"
```

```tsx
<Switch />
```
