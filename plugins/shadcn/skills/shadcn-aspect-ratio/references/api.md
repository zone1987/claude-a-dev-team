# Aspect Ratio — API Reference

## AspectRatio

| Prop | Type | Default | Required |
|------|------|---------|----------|
| `ratio` | `number` | — | Yes |
| `className` | `string` | — | No |

Common ratios: `16/9` (widescreen), `1/1` (square), `9/16` (portrait),
`4/3` (classic).

## data-slot values
