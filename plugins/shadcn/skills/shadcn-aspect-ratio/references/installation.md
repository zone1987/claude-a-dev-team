# Aspect Ratio — Installation

## CLI

```bash
npx shadcn@latest add aspect-ratio
```

## Manual — Radix UI

```bash
npm install radix-ui
```

Copy `components/ui/aspect-ratio.tsx` (Radix variant, see `source.md`).

## Manual — Base UI

```bash
npm install @base-ui/react
```

Copy `components/ui/aspect-ratio.tsx` (Base UI variant, see `source.md`).

Update import paths to match your project.
