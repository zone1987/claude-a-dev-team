# Checkbox — Installation

## CLI (recommended)

```bash
npx shadcn@latest add checkbox
```

## Manual

1. Install dependencies:

```bash
npm install radix-ui
```

2. Copy the source from `references/source.md` into `components/ui/checkbox.tsx`.
3. Update the import paths to match your project setup.

- Radix UI Checkbox docs: https://www.radix-ui.com/docs/primitives/components/checkbox
- Radix UI Checkbox API: https://www.radix-ui.com/docs/primitives/components/checkbox#api-reference
