# shadcn/ui Sonner

An opinionated toast component for React, built on top of the `sonner` library by emilkowalski.
Integrates with `next-themes` for automatic dark/light mode.

## Quick Reference

- **Install**: `npx shadcn@latest add sonner`
- **Deps**: `sonner`, `next-themes`
- **Exports**: `Toaster`
- **Usage**: `import { toast } from "sonner"` then call `toast("message")`
- **Place `<Toaster />` once** in your root layout
