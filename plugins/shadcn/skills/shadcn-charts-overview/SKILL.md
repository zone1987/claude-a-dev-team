# shadcn/ui Chart System – Overview

The shadcn chart system wraps [Recharts](https://recharts.org) with a set of
React components that handle theming, tooltips, and legends consistently.

## Installation

```bash
npx shadcn@latest add chart
```

This adds `components/ui/chart.tsx` to your project. Recharts is installed as a
peer dependency:

```bash
npm install recharts
```

## Core Components

| Component | Description |
|---|---|
| `ChartContainer` | Wrapper that provides `ChartConfig` context and renders a `ResponsiveContainer` |
| `ChartStyle` | Injects CSS custom properties derived from `ChartConfig` into the DOM |
| `ChartTooltip` | Re-export of Recharts `Tooltip` |
| `ChartTooltipContent` | Styled tooltip body that reads from `ChartConfig` |
| `ChartLegend` | Re-export of Recharts `Legend` |
| `ChartLegendContent` | Styled legend body that reads from `ChartConfig` |

## ChartConfig Type

```ts
type ChartConfig = Record<
  string,
  {
    label?: React.ReactNode
    icon?: React.ComponentType
  } & (
    | { color?: string; theme?: never }
    | { color?: never; theme: Record<"light" | "dark", string> }
  )
>
```

Fields per entry:

| Field | Type | Description |
|---|---|---|
| `label` | `React.ReactNode` | Human-readable label used in tooltip and legend |
| `icon` | `React.ComponentType` | Optional icon rendered instead of color swatch |
| `color` | `string` | CSS value e.g. `"var(--chart-1)"` or `"hsl(220 70% 50%)"` |
| `theme` | `{ light: string; dark: string }` | Per-theme color — mutually exclusive with `color` |

## ChartContainer

```tsx
<ChartContainer
  config={chartConfig}          // required: ChartConfig
  id="my-chart"                 // optional: used for CSS scoping
  className="aspect-video"      // optional: Tailwind classes
  initialDimension={{ width: 320, height: 200 }}  // optional: SSR hint
>
  {/* any Recharts chart component */}
</ChartContainer>
```

`ChartContainer` wraps everything in a `<div data-chart="chart-{id}">` and
injects a `<style>` block with `--color-{key}` custom properties for every
entry in `ChartConfig` that has a `color` or `theme`. Recharts'
`ResponsiveContainer` is rendered as the direct child.

## CSS Theming

The chart system uses CSS custom properties scoped to the chart element:

```css
/* Generated by ChartStyle for [data-chart=chart-abc] */
[data-chart=chart-abc] {
  --color-desktop: var(--chart-1);
  --color-mobile:  var(--chart-2);
}
.dark [data-chart=chart-abc] {
  --color-desktop: hsl(220 70% 50%);
  --color-mobile:  hsl(160 60% 45%);
}
```

In your Tailwind/CSS theme, provide the base palette:

```css
:root {
  --chart-1: oklch(0.646 0.222 41.116);
  --chart-2: oklch(0.6 0.118 184.714);
  --chart-3: oklch(0.398 0.07 227.392);
  --chart-4: oklch(0.828 0.189 84.429);
  --chart-5: oklch(0.769 0.188 70.08);
}
.dark {
  --chart-1: oklch(0.488 0.243 264.376);
  --chart-2: oklch(0.696 0.17 162.48);
  --chart-3: oklch(0.769 0.188 70.08);
  --chart-4: oklch(0.627 0.265 303.9);
  --chart-5: oklch(0.645 0.246 16.439);
}
```

Reference colors inside Recharts props as `var(--color-{key})`.

## ChartTooltip / ChartTooltipContent

```tsx
<ChartTooltip
  cursor={false}
  content={
    <ChartTooltipContent
      indicator="dot"            // "dot" | "line" | "dashed" (default: "dot")
      hideLabel={false}          // hide the header label
      hideIndicator={false}      // hide the color indicator
      labelFormatter={(v, payload) => String(v)}  // custom label render
      formatter={(value, name, item, index, payload) => <>{/* jsx */}</>}
      nameKey="browser"          // key in payload used to look up config entry
      labelKey="month"           // key in payload used to look up label
    />
  }
/>
```

`ChartTooltipContent` reads from `ChartConfig` context to resolve labels and
colors automatically.

## ChartLegend / ChartLegendContent

```tsx
<ChartLegend
  content={
    <ChartLegendContent
      nameKey="dataKey"    // override key for config lookup
      hideIcon={false}     // show only label, no icon/swatch
    />
  }
/>
```

Place `<ChartLegend>` inside the Recharts chart component, just like any other
Recharts child.

## Minimal Usage Example

```tsx
"use client"

import { Bar, BarChart, CartesianGrid, XAxis } from "recharts"
import {
  ChartContainer,
  ChartTooltip,
  ChartTooltipContent,
  type ChartConfig,
} from "@/components/ui/chart"

const data = [
  { month: "Jan", value: 186 },
  { month: "Feb", value: 305 },
  { month: "Mar", value: 237 },
]

const config = {
  value: { label: "Value", color: "var(--chart-1)" },
} satisfies ChartConfig

export function MyChart() {
  return (
    <ChartContainer config={config} className="h-48 w-full">
      <BarChart data={data}>
        <CartesianGrid vertical={false} />
        <XAxis dataKey="month" tickLine={false} axisLine={false} />
        <ChartTooltip content={<ChartTooltipContent />} />
        <Bar dataKey="value" fill="var(--color-value)" radius={4} />
      </BarChart>
    </ChartContainer>
  )
}
```
