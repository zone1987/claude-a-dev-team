# shadcn/ui Resizable

Accessible resizable panel groups and layouts with keyboard support, built on `react-resizable-panels` v4.

## Quick Reference

- **Install**: `npx shadcn@latest add resizable`
- **Dependency**: `react-resizable-panels` (v4+)
- **Exports**: `ResizablePanelGroup`, `ResizablePanel`, `ResizableHandle`
- **"use client"** required
