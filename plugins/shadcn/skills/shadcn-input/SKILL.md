# shadcn-input

The `Input` component is a styled HTML `<input>` with consistent focus ring, error state (`aria-invalid`), file input support, and disabled styling. No external dependencies beyond Tailwind CSS.

## Quick start

```bash
npx shadcn@latest add input
```
