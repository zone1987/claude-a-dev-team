# Input — Installation

## CLI (recommended)

```bash
npx shadcn@latest add input
```

## Manual

No external dependencies required. The `Input` component uses only Tailwind CSS.

1. Copy `components/ui/input.tsx` into your project (see `source.md`).
2. Update import paths to match your project.
