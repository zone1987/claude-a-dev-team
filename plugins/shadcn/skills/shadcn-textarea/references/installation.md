# Textarea — Installation

## CLI (recommended)

```bash
npx shadcn@latest add textarea
```

No external primitive required.

## Manual

Copy `components/ui/textarea.tsx` (see source.md).
Update import paths to match your project.

## Usage

```tsx
import { Textarea } from "@/components/ui/textarea"
```

```tsx
<Textarea />
```
