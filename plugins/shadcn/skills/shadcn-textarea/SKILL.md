# shadcn/ui Textarea

Displays a form textarea or a component that looks like a textarea. Built on
native HTML `<textarea>` element with Tailwind styling. Supports `field-sizing-content`
for auto-resize behavior.

## Quick Reference

- **Install**: `npx shadcn@latest add textarea`
- **Deps**: none (native HTML element)
- **Exports**: `Textarea`
- **Auto-resize**: uses CSS `field-sizing-content` (min-height: 16 × 4px)
