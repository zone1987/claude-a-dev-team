# Empty — Installation

## CLI (recommended)

```bash
npx shadcn@latest add empty
```

## Manual

No external dependencies required. The `Empty` component uses only Tailwind CSS and `class-variance-authority` (already bundled with shadcn/ui).

1. Copy `components/ui/empty.tsx` into your project (see `source.md`).
2. Update import paths to match your project.
