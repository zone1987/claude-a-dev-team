# shadcn/ui — Avatar

An image element with a fallback for representing the user. Built on Radix
UI or Base UI Avatar primitives.

## Sub-components

- `Avatar` — root (`size="default" | "sm" | "lg"`)
- `AvatarImage` — image; handles loading state
- `AvatarFallback` — shown when image fails
- `AvatarBadge` — status indicator dot (bottom-right)
- `AvatarGroup` — overlapping group wrapper
- `AvatarGroupCount` — "+N" count badge in a group
