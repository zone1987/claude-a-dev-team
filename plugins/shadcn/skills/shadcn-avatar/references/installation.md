# Avatar — Installation

## CLI

```bash
npx shadcn@latest add avatar
```

## Manual — Radix UI

```bash
npm install radix-ui
```

Copy `components/ui/avatar.tsx` (Radix variant, see `source.md`).

## Manual — Base UI

```bash
npm install @base-ui/react
```

Copy `components/ui/avatar.tsx` (Base UI variant, see `source.md`).

Update import paths to match your project.
