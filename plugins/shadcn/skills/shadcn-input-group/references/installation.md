# InputGroup — Installation

## CLI (recommended)

```bash
npx shadcn@latest add input-group
```

## Manual

Requires `button`, `input`, and `textarea` components.

1. Ensure dependencies are installed:

```bash
npx shadcn@latest add button input textarea
```

2. Copy `components/ui/input-group.tsx` into your project (see `source.md`).
3. Update import paths to match your project.
