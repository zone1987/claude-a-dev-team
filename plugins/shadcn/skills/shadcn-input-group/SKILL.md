# shadcn-input-group

The `InputGroup` wraps inputs and addons in a unified focus-managed container. Addons (icons, text, buttons) can be placed at four positions via the `align` prop. `InputGroupInput` and `InputGroupTextarea` replace the plain `Input`/`Textarea` inside groups. Uses CSS `has-[]` selectors for focus and error state propagation.

## Quick start

```bash
npx shadcn@latest add input-group
```
