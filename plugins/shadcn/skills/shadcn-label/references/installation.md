# Label — Installation

## CLI

```bash
npx shadcn@latest add label
```

## Manual (Radix variant — default new-york-v4)

1. Install dependency:

```bash
npm install @radix-ui/react-label
# or via the radix-ui umbrella package:
npm install radix-ui
```

2. Copy the source into `components/ui/label.tsx`.

## Manual (Base UI variant)

1. Install dependency:

```bash
npm install @base-ui/react
```

2. Use the base variant source (see `base-vs-radix.md`).
