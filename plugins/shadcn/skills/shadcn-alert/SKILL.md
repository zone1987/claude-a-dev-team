# shadcn/ui — Alert

Displays a callout for user attention. Pure CSS component — no external
primitives required for Radix version.

## Sub-components

- `Alert` — root container (`role="alert"`)
- `AlertTitle` — heading line
- `AlertDescription` — body / detail text
- `AlertAction` — optional action slot (Base UI only in default registry)

## Variants

- `default` — card background
- `destructive` — red-tinted with destructive text color
