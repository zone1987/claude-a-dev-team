# Alert — Installation

## CLI

```bash
npx shadcn@latest add alert
```

## Manual

No external dependencies required (Radix version).

Copy `components/ui/alert.tsx` (see `source.md`).

### Base UI (manual)

```bash
npm install @base-ui/react
```

Copy the Base UI variant of `alert.tsx` (see `source.md`).

Update import paths to match your project.
