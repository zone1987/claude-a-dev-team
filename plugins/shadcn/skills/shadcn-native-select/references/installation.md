# NativeSelect — Installation

## CLI

```bash
npx shadcn@latest add native-select
```

## Manual

Copy the component source from `references/source.md` into `components/ui/native-select.tsx` and update import paths.

### Dependencies

Only requires:
- `lucide-react` for `ChevronDownIcon`
- `@/lib/utils` for the `cn` helper

```bash
npm install lucide-react
```
