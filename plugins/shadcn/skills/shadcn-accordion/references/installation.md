# Accordion — Installation

## CLI

```bash
npx shadcn@latest add accordion
```

## Manual

### Radix UI

Install dependency:

```bash
npm install radix-ui
```

Copy `components/ui/accordion.tsx` (see `source.md`).

### Base UI

Install dependency:

```bash
npm install @base-ui/react
```

Copy `components/ui/accordion.tsx` (Base variant, see `source.md`).

Update import paths to match your project.
