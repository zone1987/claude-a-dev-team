# shadcn/ui — Badge

Displays a badge or a component that looks like a badge. Uses
class-variance-authority for variants.

## Variants

`default`, `secondary`, `destructive`, `outline`, `ghost`, `link`

## Key notes

- Radix version: `asChild` prop for rendering as link/other element
- Base version: `render` prop with Base UI `useRender`
- Export `badgeVariants` for applying badge styling to non-Badge elements
- Use `data-icon="inline-start"` / `data-icon="inline-end"` on icons inside
