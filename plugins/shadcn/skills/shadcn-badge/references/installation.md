# Badge — Installation

## CLI

```bash
npx shadcn@latest add badge
```

## Manual

No external primitive dependency for Radix version.

```bash
npm install radix-ui  # only needed for Slot in Radix version
```

Copy `components/ui/badge.tsx` (see `source.md`).

### Base UI

```bash
npm install @base-ui/react
```

Copy the Base UI variant of `badge.tsx` (see `source.md`).

Update import paths to match your project.
