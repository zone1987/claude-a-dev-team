# shadcn-dialog

The `Dialog` component creates a modal window overlaid on the primary window. Built on Radix UI (new-york-v4) or Base UI. Supports optional close button, sticky footer, scrollable content, and RTL.

## Sub-components

`Dialog` · `DialogTrigger` · `DialogPortal` · `DialogOverlay` · `DialogClose` · `DialogContent` · `DialogHeader` · `DialogFooter` · `DialogTitle` · `DialogDescription`

## Quick start

```bash
npx shadcn@latest add dialog
```
