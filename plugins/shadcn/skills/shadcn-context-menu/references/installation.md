# ContextMenu — Installation

## CLI (recommended)

```bash
npx shadcn@latest add context-menu
```

## Manual

1. Install dependencies:

```bash
npm install radix-ui
```

2. Copy the source from `references/source.md` into `components/ui/context-menu.tsx`.
3. Update the import paths to match your project setup.

- Radix UI ContextMenu docs: https://www.radix-ui.com/docs/primitives/components/context-menu
