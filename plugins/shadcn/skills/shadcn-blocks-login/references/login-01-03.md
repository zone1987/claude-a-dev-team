# Login Blocks 01–03

## login-01

Simple centered card with email, password, Google login button.

**Install:** `npx shadcn@latest add login-01`

**File tree:**
```
login-01/
├── page.tsx
└── components/
    └── login-form.tsx
```

### login-01/page.tsx

```tsx
import { LoginForm } from "@/registry/new-york-v4/blocks/login-01/components/login-form"

export default function Page() {
  return (
    <div className="flex min-h-svh w-full items-center justify-center p-6 md:p-10">
      <div className="w-full max-w-sm">
        <LoginForm />
      </div>
    </div>
  )
}
```
