# Carousel — Installation

## CLI (recommended)

```bash
npx shadcn@latest add carousel
```

## Manual

1. Install dependencies:

```bash
npm install embla-carousel-react
```

2. Copy the source from `references/source.md` into `components/ui/carousel.tsx`.
3. Update the import paths to match your project setup.

## Links

- Embla Carousel docs: https://www.embla-carousel.com/get-started/react
- Embla Carousel API: https://www.embla-carousel.com/api
