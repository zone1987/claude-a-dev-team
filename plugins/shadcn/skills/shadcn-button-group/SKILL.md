# shadcn/ui — Button Group

A container that groups related buttons together with consistent styling and
shared border treatment.

## Sub-components

- `ButtonGroup` — root container (`orientation="horizontal" | "vertical"`)
- `ButtonGroupSeparator` — visual divider between buttons
- `ButtonGroupText` — static text label within the group

## Nesting

Nest `<ButtonGroup>` inside another `<ButtonGroup>` to create groups with
gap spacing between sections.
