# Button Group — Installation

## CLI

```bash
npx shadcn@latest add button-group
```

## Manual — Radix UI

```bash
npm install radix-ui
```

Requires `Separator` component. Copy `components/ui/button-group.tsx` (see `source.md`).

## Manual — Base UI

```bash
npm install @base-ui/react
```

Requires `Separator` component. Copy Base UI variant of `button-group.tsx` (see `source.md`).

Update import paths to match your project.
