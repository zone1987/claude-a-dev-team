# shadcn/ui Toggle

A two-state button that can be either on or off. Built on Radix UI Toggle primitive.
Exports `toggleVariants` for use in custom components (e.g. ToggleGroup).

## Quick Reference

- **Install**: `npx shadcn@latest add toggle`
- **Radix deps**: `radix-ui`
- **Exports**: `Toggle`, `toggleVariants`
- **Variants**: `default` | `outline`
- **Sizes**: `default` (h-9 min-w-9) | `sm` (h-8 min-w-8) | `lg` (h-10 min-w-10)
