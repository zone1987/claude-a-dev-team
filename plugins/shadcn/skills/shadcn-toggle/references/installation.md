# Toggle — Installation

## CLI (recommended)

```bash
npx shadcn@latest add toggle
```

## Manual

Install dependency:

```bash
npm install radix-ui
```

Copy `components/ui/toggle.tsx` (see source.md).
Update import paths to match your project.

## Usage

```tsx
import { Toggle } from "@/components/ui/toggle"
```

```tsx
<Toggle>Toggle</Toggle>
```
