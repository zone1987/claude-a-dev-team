# shadcn/ui Sheet

Extends the Dialog component to display content that complements the main content of
the screen. Slides in from any edge (`top`, `right`, `bottom`, `left`).

## Quick Reference

- **Install**: `npx shadcn@latest add sheet`
- **Radix dep**: `radix-ui` (Dialog) | **Base dep**: `@base-ui/react` (Dialog)
- **Exports**: `Sheet`, `SheetTrigger`, `SheetClose`, `SheetContent`, `SheetHeader`,
  `SheetFooter`, `SheetTitle`, `SheetDescription`
