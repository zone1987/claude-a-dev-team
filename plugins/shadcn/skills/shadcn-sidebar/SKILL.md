# shadcn/ui Sidebar

A composable, themeable, and customizable sidebar component with collapsible modes,
mobile sheet fallback, keyboard shortcut, and full CSS variable theming.

## Quick Reference

- **Install**: `npx shadcn@latest add sidebar`
- **Keyboard shortcut**: `Cmd+B` / `Ctrl+B`
- **CSS width vars**: `--sidebar-width` (default `16rem`), `--sidebar-width-icon` (`3rem`)
- **Collapsible modes**: `offcanvas`, `icon`, `none`
- **Variants**: `sidebar`, `floating`, `inset`
