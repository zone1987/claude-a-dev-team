# Direction — Installation

## CLI (recommended)

```bash
npx shadcn@latest add direction
```

## Manual — Radix UI (new-york-v4 default)

1. Install dependency:

```bash
npm install radix-ui
```

2. Copy `components/ui/direction.tsx` into your project.
3. Update import paths to match your project.

## Manual — Base UI

1. Install dependency:

```bash
npm install @base-ui/react
```

2. Copy the Base UI variant (single re-export line) into your project.
