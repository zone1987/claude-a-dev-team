# Alert Dialog — Installation

## CLI

```bash
npx shadcn@latest add alert-dialog
```

## Manual — Radix UI

```bash
npm install radix-ui
```

Copy `components/ui/alert-dialog.tsx` (Radix variant, see `source.md`).

## Manual — Base UI

```bash
npm install @base-ui/react
```

Copy `components/ui/alert-dialog.tsx` (Base UI variant, see `source.md`).

Update import paths to match your project.
