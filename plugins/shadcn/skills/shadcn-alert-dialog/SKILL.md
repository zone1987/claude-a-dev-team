# shadcn/ui — Alert Dialog

A modal dialog that interrupts the user with important content and expects
a response. Built on Radix UI or Base UI Alert Dialog primitives.

## Sub-components

- `AlertDialog` — root
- `AlertDialogTrigger` — opens the dialog
- `AlertDialogContent` — dialog panel (`size="default" | "sm"`)
- `AlertDialogHeader` — top section
- `AlertDialogMedia` — icon/image area in header
- `AlertDialogTitle`, `AlertDialogDescription` — text content
- `AlertDialogFooter` — action row
- `AlertDialogAction`, `AlertDialogCancel` — buttons (wrap Button)
