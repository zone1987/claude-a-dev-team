# Command — Installation

## CLI (recommended)

```bash
npx shadcn@latest add command
```

## Manual

1. Install dependencies:

```bash
npm install cmdk
```

2. Copy the source from `references/source.md` into `components/ui/command.tsx`.
3. Update the import paths to match your project setup.
