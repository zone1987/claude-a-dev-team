# shadcn/ui Separator

Visually or semantically separates content. Available in Radix UI and Base UI variants.

## Quick Reference

- **Install**: `npx shadcn@latest add separator`
- **Radix dep**: `radix-ui` | **Base dep**: `@base-ui/react`
- **Exports**: `Separator`
- Default orientation: `horizontal`
