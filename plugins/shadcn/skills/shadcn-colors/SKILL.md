# shadcn/ui — Colors Reference

shadcn/ui uses OKLCH for all color values in Tailwind v4.

## Available base colors for `tailwind.baseColor`

These are the shadcn/ui semantic base colors (used for the neutral/grey tones
in your theme):

| baseColor | Character |
|-----------|-----------|
| `neutral` | Pure grey, no hue (default) |
| `stone` | Warm grey with slight brown |
| `zinc` | Slightly blue-toned grey |
| `mauve` | Purple-tinted grey |
| `olive` | Green-tinted grey |
| `mist` | Cool blue-grey |
| `taupe` | Warm beige-grey |

## Tailwind base color palettes (available via index.json)

Full palette colors (scales 50–950): slate, gray, zinc, neutral, stone, red,
orange, amber, yellow, lime, green, emerald, teal, cyan, sky, blue, indigo,
violet, purple, fuchsia, pink, rose.

API endpoint: `https://ui.shadcn.com/r/colors/index.json`
