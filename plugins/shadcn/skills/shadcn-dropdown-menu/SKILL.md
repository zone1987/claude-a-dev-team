# shadcn-dropdown-menu

The `DropdownMenu` component displays a contextual menu triggered by a button. Built on Radix UI. Supports labels, separators, checkboxes, radio groups, submenus, keyboard shortcuts, icons, and destructive variants.

## Quick start

```bash
npx shadcn@latest add dropdown-menu
```
