# shadcn-field

The `Field` family composes accessible form fields. `Field` provides orientation variants, `FieldError` handles validation (supports `errors` array from react-hook-form and Standard Schema), `FieldSeparator` divides sections, and `FieldSet`/`FieldLegend` wrap semantically grouped fields.

## Quick start

```bash
npx shadcn@latest add field
```
