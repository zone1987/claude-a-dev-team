# Field — Installation

## CLI (recommended)

```bash
npx shadcn@latest add field
```

## Manual

No external dependencies required beyond shadcn/ui itself (`Label`, `Separator` components must be installed).

1. Ensure `label` and `separator` are installed:

```bash
npx shadcn@latest add label separator
```

2. Copy `components/ui/field.tsx` into your project (see `source.md`).
3. Update import paths to match your project.
