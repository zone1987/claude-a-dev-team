# Kbd — Installation

## CLI

```bash
npx shadcn@latest add kbd
```

## Manual

Copy the component source from `references/source.md` into `components/ui/kbd.tsx` and update import paths.

### Dependencies

The `Kbd` component only requires:
- `@/lib/utils` for the `cn` helper

No additional npm packages are needed.
