# RadioGroup — Installation

## CLI

```bash
npx shadcn@latest add radio-group
```

## Manual (Radix variant — new-york-v4 default)

1. Install dependency:

```bash
npm install radix-ui
# or specifically:
npm install @radix-ui/react-radio-group
```

2. Copy the source into `components/ui/radio-group.tsx`.

## Manual (Base UI variant)

1. Install dependency:

```bash
npm install @base-ui/react
```

2. Use the base variant source (see `base-vs-radix.md`).
