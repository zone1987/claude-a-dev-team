# Combobox — Installation

## CLI (recommended)

```bash
npx shadcn@latest add combobox
```

## Manual

1. Install dependencies:

```bash
npm install @base-ui/react
```

2. Copy the source from `references/source.md` into `components/ui/combobox.tsx`.
3. Update the import paths to match your project setup.

## Links

- Base UI Combobox docs: https://base-ui.com/react/components/combobox
- Base UI Combobox API: https://base-ui.com/react/components/combobox#api-reference

## Note on the Classic (Command-based) Pattern

The examples in `references/examples.md` also show the older pattern of building a combobox from `Command` + `Popover`. Both approaches are valid. The Base UI `Combobox` component (this skill) is the newer, dedicated approach.
