# shadcn/ui Tabs

A set of layered sections of content (tab panels) displayed one at a time.
Built on Radix UI Tabs primitive. Supports horizontal/vertical orientation and
default/line list variants.

## Quick Reference

- **Install**: `npx shadcn@latest add tabs`
- **Radix deps**: `radix-ui`
- **Exports**: `Tabs`, `TabsList`, `TabsTrigger`, `TabsContent`, `tabsListVariants`
- **Orientation**: `horizontal` (default) | `vertical`
- **TabsList variants**: `default` | `line`

## Composition

```text
Tabs
├── TabsList
│   ├── TabsTrigger
│   └── TabsTrigger
├── TabsContent
└── TabsContent
```
