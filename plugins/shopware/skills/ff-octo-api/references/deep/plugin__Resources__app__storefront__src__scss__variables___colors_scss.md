# SCSS variables/_colors.scss (`.../scss/variables/_colors.scss`)

## Zweck
Farb-Variablen gemäß Figma-Styleguide: Neutrals (`$white`, `$black`, `$gray-*`), Corporate-/Location-Farben (`$primary-london` #D92127, `$primary-new-york`, `$primary-dubai`), `$primary` (= London), `$secondary` (#27346B).

## Typ
- SCSS-Variablen-Datei.

## Besonderheiten
- `$primary` ist standardmäßig London-Rot → location-spezifisches Theming via diese Variablen.

## Bezüge
`base.scss`, `component/_flatpickr.scss`, `component/_product-detail-configurator.scss`.
