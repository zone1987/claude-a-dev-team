# SCSS variables/_custom.scss (`.../scss/variables/_custom.scss`)

## Zweck
Definiert Spacing- und Typografie-Variablen gemäß Figma-Styleguide (Abstände `$spacing-space-*`, Font-Sizes `$font-size-*` inkl. Mobile/XS-Varianten, Zeilenhöhen).

## Typ
- SCSS-Variablen-Datei.

## Besonderheiten
- Eigene Variablen — können sich mit Theme-Variablen überschneiden; bei Theme-Anpassungen abgleichen.

## Bezüge
`base.scss`, `component/*.scss`.
