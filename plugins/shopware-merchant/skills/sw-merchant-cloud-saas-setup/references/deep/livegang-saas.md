# Shopware SaaS — Livegang

**Quelle**: https://docs.shopware.com/de/shopware-6-de/saas/livegang

---

## Schritt 1: Plan wählen

- 30-tägige Testphase vorab (kein Plan nötig)
- Optionen: **Shopware Rise**, **Shopware Evolve**, **Shopware Beyond**
- Alternative: self-hosted Shop

---

## Schritt 2: Umsatzsteuer-Identifikationsnummer

**Pfad:** Einstellungen > Firma

Optionen:
- „Ich besitze eine Umsatzsteuer-ID" → ID eingeben
- „Ich besitze keine Umsatzsteuer-ID"
- Optional: „Ich bin Kleingewerbetreibender"

> ⚠️ **Achtung:** Nicht mit der **Steuernummer** oder der **Schweizer UID** verwechseln!

---

## Schritt 3: Rechnungsinformationen

**Pfad:** Einstellungen > Firma > Rechnungsinformationen

| Feld | Pflichtfeld |
|---|---|
| Firmenname | ✓ (Sternchen) |
| Branche | ✓ (Dropdown, inkl. „Weiteres") |
| E-Mail | ✓ |
| E-Mail für Mahnungen | Nein (optional) |
| Straße + Hausnummer | ✓ |
| PLZ | ✓ |
| Stadt | ✓ |
| Land | ✓ (Dropdown) |

---

## Schritt 4: PayPal verbinden

**Pfad:** Einstellungen > Account > Zahlungsart

1. Klick auf **„Neues PayPal Konto verbinden"**
2. Weiterleitung zu PayPal (bestehenden Account einloggen oder neuen erstellen)
3. Verbindung bestätigen

---

## Schritt 5: Übersicht (nach PayPal-Verbindung)

Nach erfolgreicher Einrichtung zeigt die Übersicht:

| Element | Inhalt |
|---|---|
| (1) Firma | Kontakt- und Rechnungsinformationen |
| (2) Gewählter Plan | Funktionsumfang und Kosten |
| (3) Zahlungsart | PayPal als Standard |
| (4) Shopware Account Einstellungen | Weiterleitung zum Account |

---

## Was muss ich noch beachten? (Checkliste)

### Rechtstexte pflegen
- **Pfad:** Inhalte > Erlebniswelten
- Vordefinierte Seiten (mit Blindtext):
  - Impressum
  - Datenschutz
  - Widerrufsbelehrungen
  - AGB
- Bearbeitung: In Text klicken > Speichern (rechts oben)
- Verwaltung: Kataloge > Kategorien > Footer Menü > Rechtliches

### Produkte pflegen
- **Pfad:** Kataloge > Produkte
- Button „Produkt anlegen" (rechts oben)

### Eigenes Theme einrichten
- **Menüpunkt:** Themes
- Theme auswählen und anpassen

### Versandarten pflegen
- **Pfad:** Einstellungen > Shop > Versand
- „Versandart anlegen" (rechts oben)

### Zahlungsarten pflegen
- **Pfad:** Einstellungen > Shop > Zahlungsarten
- Standardzahlungsarten: Nachnahme, Vorkasse, Rechnung
