# Shopware CAD to 3D File Conversion

Konvertiert CAD-Dateien im STEP-Format in webfähige GLB-3D-Modelle direkt im Media Manager.
Die Konvertierung läuft im Hintergrund.

**Pfad:** Inhalte > Medien → Datei auswählen → „In 3D-Modell konvertieren"

**Mindestversion:** Shopware 6.7.6.0 | Freikontingent: 1 Konvertierung/Monat
