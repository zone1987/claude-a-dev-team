# Shopware 6 – Social Shopping

Produkt-Feeds für Social-Media-Plattformen (ab Shopware Rise).

## Unterstützte Plattformen

| Plattform | Feed-Typ | Besonderheit |
|---|---|---|
| **Facebook** | XML | Referral-Code für Tracking |
| **Instagram** | XML | Identisch zu Facebook |
| **Google Shopping** | XML | Domain-Verifizierung erforderlich |
| **Pinterest** | Kein Feed | Meta-Daten / Rich Pins |

## Grundprinzip

1. Social-Shopping-Kanal anlegen (Typ wählen)
2. Storefront-Kanal + Domain zuweisen
3. Dynamische Produktgruppe für Produktauswahl
4. Export-URL in Plattform eintragen
5. Template anpassen (optional, Twig)

## Voraussetzung

Shopware Rise (oder höher) Plan; Extension unter Erweiterungen > Meine Erweiterungen aktivieren.
