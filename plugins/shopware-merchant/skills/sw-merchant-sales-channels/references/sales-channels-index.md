## Enthaltene Skills

| Skill | Thema |
|---|---|
| `sw-merchant-sales-channels` | Überblick, Typen, Favoriten, Kunden-Bindung |
| `sw-merchant-sales-channels-create` | Kanal anlegen, Typ wählen |
| `sw-merchant-sales-channels-config` | Grundeinstellungen, Navigation, Zahlung, Analyse |
| `sw-merchant-sales-channels-domains` | Domains, Sprachen, Währungen, Hreflang |
| `sw-merchant-sales-channels-products` | Produktzuweisung über Kanal-Reiter |
| `sw-merchant-sales-channels-api-access` | API-Schlüssel generieren und nutzen |
| `sw-merchant-sales-channels-social-shopping` | Facebook, Instagram, Google Shopping, Pinterest |
| `sw-merchant-sales-channels-product-comparison` | XML/CSV-Feeds, Twig-Templates, Variablen |
| `sw-merchant-sales-channels-agentic` | JSONL-Feed für KI-Plattformen (ab 6.7.10) |
