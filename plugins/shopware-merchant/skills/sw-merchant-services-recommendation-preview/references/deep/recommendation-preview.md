# Shopware Empfehlungs-Vorschau — Vollständige Referenz

Quelle: https://docs.shopware.com/de/shopware-6-de/shopware-services/empfehlungs-vorschau

---

## Was ist die Empfehlungs-Vorschau?

Der Empfehlungs-Vorschau-Service präsentiert **intelligente, kontextbezogene Empfehlungen**
direkt in der Administrationsoberfläche. Händler werden proaktiv über Features und Erweiterungen
informiert, die den Shop-Betrieb verbessern könnten — ohne den Workflow zu unterbrechen.

## Mindestvoraussetzungen

- Shopware **6.7.1.0** oder neuer
- Aktiver Shopware-Plan (für Empfehlungen ausblenden)

## Funktionen

### Kontextuelle Platzierung
Empfehlungen erscheinen an **relevanten Stellen** innerhalb des Admin-Bereichs.
Der Service analysiert den Kontext und zeigt passende Vorschläge zur richtigen Zeit.

### Visuelle Komponenten
Jede Empfehlung enthält:
- **Icon** — visuelles Erkennungszeichen
- **Kurzer Titel** — prägnante Bezeichnung
- **Beschreibung** — kurze Erläuterung des Nutzens
- **Link** — zu weiteren Informationen oder direkte Installation

### Empfehlungen ausblenden
Händler mit einem **aktiven Shopware-Plan** können:
- Einzelne Empfehlungen über das **Kontextmenü** ausblenden
- Einmal ausgeblendet: keine weiteren Empfehlungen für diese Erweiterung

## Einrichtung

Keine manuelle Einrichtung erforderlich. Empfehlungen werden **automatisch** in der
Administration angezeigt.

## Nutzung

1. Empfehlung erscheint automatisch im Admin-Bereich
2. Kurzbeschreibung lesen
3. **Option A:** Auf den Link klicken für weitere Informationen
4. **Option B:** Erweiterung direkt über den Link installieren
5. **Option C:** Empfehlung über Kontextmenü ausblenden (nur mit aktivem Shopware-Plan)

## Abgrenzung

| Aspekt | Empfehlungs-Vorschau |
|---|---|
| Ziel | Proaktive Feature-Entdeckung |
| Steuerung durch Händler | Ausblenden einzelner Empfehlungen möglich |
| Kostenpflichtig | Nein (Service kostenlos) |
| Ausblenden erfordert | Aktiven Shopware-Plan |
