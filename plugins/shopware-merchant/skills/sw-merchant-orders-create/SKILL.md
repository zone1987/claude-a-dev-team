# Shopware 6 – Bestellung manuell im Admin anlegen

Anleitung zum Erstellen manueller Bestellungen über die Shopware-Administration.

Ausführliche Dokumentation: [`references/deep/create.md`](references/deep/create.md)

## Schnellübersicht

1. Bestellungen > „Bestellung anlegen"
2. Kunden auswählen oder neu anlegen
3. Produkte / leere Positionen / Gutschriften hinzufügen
4. Optionen konfigurieren (Zahlung, Versand, Adressen, Währung)
5. „Bestellung speichern" → Bestätigungs-E-Mail wird automatisch versendet
